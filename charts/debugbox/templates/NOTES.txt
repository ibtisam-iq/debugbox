Thank you for installing DebugBox.

To launch a debug pod:
  kubectl run debug --rm -it --image={{ .Values.image.repository }}:{{ .Values.image.tag }} -- bash

To enable daemonset:
  helm upgrade --install debugbox charts/debugbox --set daemonset.enabled=true

To deploy privileged:
  helm upgrade --install debugbox charts/debugbox --set pod.privileged=true,pod.hostNetwork=true,rbac.clusterWide=true
